Projects

Most of my projects can be found on my github profile.

Sometime I might update this site to show them here, but for now I'll only tell you about this interesting project which I've built for showcasing software testing for a master course at the university.
It uses a simple Spring Boot app which exposes some endpoints. The main thing with the project is the GitHub Actions integration for which you can see a sample run here. It uses Cucumber testing with the Gherkin syntax which I've found to be one of the best of to implement unit tests in Java (and also make anyone understand what you're doing there as it uses the natural language for BDD - Behaviour Driven Development).
